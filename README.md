## 两种搜索模式，各有用处

TextSeek 提供了两种搜索模式，适合不同场景：

**简易模式**：不建索引，点击搜索后直接扫描文件夹。适合临时搜索或不固定的目录，缺点是每次搜索都要重新扫，速度相对慢一点。好处是开箱即用，上手零门槛。
